Les Variables en JavaScript

En JavaScript, il existe trois façons de déclarer une variable, chacune peut stocker n'importe quel type de données :

  • const : C'est une variable à laquelle nous ne pourrons jamais changer sa valeur.
  • let : C'est une variable à laquelle nous pourrons changer sa valeur dans le futur.
  • var : C'est une variable à laquelle nous pourrons changer sa valeur dans le futur.

Étapes de création d'une variable :

  1. Ajout du mot clé de la variable :
    • const : Déclare une variable avec une valeur constante.
      •        const nomDeVariable = valeur;
    • let : Déclare une variable avec une valeur modifiable.
      •        let nomDeVariable = valeur;
    • var : Déclare une variable avec une portée globale ou de fonction.
      •        var monNomDeVariable = valeur;
  2. Le nom de la variable :
    • camelCase : Notation où la première lettre de chaque mot (sauf le premier) est en majuscule.
      •        let produitVaisselle;
    • snake_case : Notation où les mots sont séparés par des underscores "_".
      •        let _produit;
             let produit_vaisselle;
  3. Le signe d'assignement :
    • le signe égal : Utilisé pour assigner une valeur à une variable.
      •        let _produit = "MIR";
             let prix_produit = 10;
  4. La valeur de la variable :
    • données : Peut être un tableau, un nombre, un booléen, une chaîne de caractères, etc.
      •        let _produit = "MIR";
             let prix_produit = 10;

Utilisation d'une variable const

La variable const est utilisée pour déclarer des valeurs constantes qui ne peuvent pas être modifiées une fois assignées. Pour les objets et les tableaux déclarés avec const, leurs propriétés ou éléments peuvent cependant être modifiés.

Utilisation d'une variable let

La variable let est utilisée pour déclarer des variables dont la valeur peut être modifiée ultérieurement. Contrairement à const, les variables déclarées avec let peuvent être réassignées.
