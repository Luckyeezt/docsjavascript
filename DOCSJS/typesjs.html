Les différents types de données en Javascript

  • String : C'est une séquence de caractères comme des lettres, chiffres, symboles pour représenter un texte en Javascript.
  • Boolean : Permet de représenter une valeur logique avec les booléens true(vrai) et false(faux).
  • Number : Permet de réprésenter des valeurs numériques comme un nombre entier ou un nombre décimal.
  • Array : Permet de stocker une collection ordonnée d'élements, offre une manipulation de données efficace telle qu'une liste.
  • Object : Permet de regrouper des données et des fonctionnalités dans une seule unité.
  • undefined : Valeur par défaut si une variable à été déclarée mais aucune valeur lui à été assignée.
  • null : Réprésente l'absence volontaire de valeur pour une variable.

Le type : String (chaine de caractères)

Un "string" est une chaine de caractères, cela peut être un mot, une expression ou une phrase.

Comme tout élément en Javascript, le string à 3 syntaxes particulières (3 façons de s'écrire) :


  • Guillemets doubles : -> Utilisés pour créer une chaîne de caractères.
  • Guillemets simples : -> Utilisés également pour créer une chaîne de caractères.
  • Backticks : -> Utilisés pour créer une chaïne de caractères sur plusieurs lignes (évitant la concaténation) et est utilisé pour les interpolations de variables.
"Bienvenue sur notre site !" // Ici on utilise les guillemets doubles.
'Bienvenue sur notre site !' // Ici on utilise les guillemets simples.
`Bienvenue sur notre site !` // Ici on utilise les backticks.

Le type : Number (valeurs numériques)

Un "number" est une valeur numérique en JavaScript, pouvant être un entier ou un nombre décimal.

Les nombres s'utilisent principalement pour les opérations, en Javascript nous pouvons écrire un nombre entier et décimal qu'il soit positif ou négatif :

  2.5 // Ceci est un nombre décimal positif.
  1000000 // Ceci est un nombre entier positif.
  -50.25 // Ceci est un nombre décimal négatif.
  -100 // Ceci est un nombre entier négatif.

Le type : Boolean (instructions logiques)

Un "boolean" contient seulement deux valeurs "true" (vrai) et "false" (faux)

Les booléens s'utilisent comme des conditions vraies ou fausses, principalement utilisés pour vérifier si quelque chose est vrai ou faux (avec une comparaison par exemple), dans ce cas true reflète "oui" et false reflète "non"
