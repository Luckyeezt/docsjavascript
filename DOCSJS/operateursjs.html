Les Opérateurs Arithmétiques en JavaScript

JavaScript offre plusieurs opérateurs arithmétiques pour effectuer des calculs numériques. Voici les principaux opérateurs :

  • + : Addition
  • - : Soustraction
  • * : Multiplication
  • / : Division
  • % : Modulo (reste de la division)
  • ** : Puissance (exposant)

Opérateurs d'Assignation Combinés

En JavaScript, il existe des opérateurs d'assignation combinés qui permettent de raccourcir les expressions :

let value = 10;

  • value += 5; // équivalent à value = value + 5; (value devient 15)
  • value -= 3; // équivalent à value = value - 3; (value devient 12)
  • value *= 2; // équivalent à value = value * 2; (value devient 24)
  • value /= 4; // équivalent à value = value / 4; (value devient 6)
  • value **= 2; // équivalent à value = value ** 2; (value devient 36)
  • value %= 5; // équivalent à value = value % 5; (value devient 1)
